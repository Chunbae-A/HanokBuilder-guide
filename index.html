한국 전통 건축 3D 체험 프로그램 가이드

이 문서는 한옥과 전통 건축 요소를 3D 공간에서 살펴보고, 모듈 라이브러리의 부재와 소품을 직접 배치해보는 사용자를 위한 공식 안내서입니다.

목적
전통 건축 요소를 공간 안에서 관찰하고 배치하며 구조와 분위기를 이해합니다.
대상
한국 전통 건축을 처음 접하는 사용자, 교육용 체험 자료가 필요한 사용자입니다.
기획 의도
텍스트와 이미지로는 전달하기 어려운 배치 관계를 직접 조작 가능한 3D 장면으로 보여줍니다.

Interface Guide

화면 구성

실행 화면은 에셋을 고르는 영역, 장면을 보는 영역, 선택한 항목을 편집하는 영역으로 나뉩니다. 아래 번호를 기준으로 각 영역의 역할을 확인할 수 있습니다.

  1. 모듈 라이브러리

    배치할 수 있는 전통 건축 부재, 자연 요소, 공간 소품을 썸네일로 확인합니다.

  2. 검색과 카테고리

    이름 검색 또는 카테고리 선택으로 필요한 에셋만 좁혀서 봅니다.

  3. 공간 프리셋

    한옥 마당, 사랑채, 조선 장터, 전통 정원 분위기를 전환합니다.

  4. 3D 뷰포트

    건축물과 배치한 에셋을 직접 보고, 장면 안의 항목을 선택합니다.

  5. 조작 툴바

    이동, 회전, 크기, 삭제, 초기화, 캡처 같은 빠른 작업을 실행합니다.

  6. 배치 편집 패널

    선택한 에셋의 위치, 회전, 크기를 숫자와 버튼으로 세밀하게 조정합니다.

  7. 추천 입력창

    원하는 장면을 문장으로 입력해 관련 에셋 추천과 자동 배치를 요청합니다.

시점 회전 마우스 오른쪽 드래그 또는 Alt + 왼쪽 드래그

확대와 축소 마우스 휠

Module Library

모듈 라이브러리에서 에셋 선택하기

모듈 라이브러리는 사용자가 배치할 수 있는 부재와 소품을 한곳에 모아 보여주는 영역입니다. 검색창과 카테고리 버튼을 함께 사용하면 원하는 에셋을 빠르게 찾을 수 있습니다.

  • 검색창에 이름 일부를 입력하면 목록이 좁혀집니다.
  • 카테고리를 선택하면 관련 에셋만 표시됩니다.
  • 썸네일을 클릭하면 해당 에셋이 장면에 추가됩니다.
  • 썸네일 아래 이름은 사용자가 이해하기 쉬운 표시 이름을 기준으로 보여야 합니다.

Placement Editing

배치 편집 패널 사용하기

최신 배치 편집 패널은 선택한 에셋의 이름을 먼저 보여주고, 위치와 회전 값을 축별로 조정할 수 있도록 구성되어 있습니다. 작은 -, + 버튼은 값을 조금씩 바꿀 때 사용합니다.

위치 X, Y, Z 값을 입력하거나 축별 -, + 버튼으로 좌표를 조정합니다.
회전 X, Y, Z 회전값을 조정하고, 회전 단위 선택값에 맞춰 방향을 맞춥니다.
크기 배율 입력 또는 0.5x, 1x, 2x, 3x 버튼으로 크기를 조절합니다.
빠른 작업 복제, 삭제, 선택 해제를 실행합니다.

입력 필드는 Tab 키로 다음 항목으로 이동할 수 있습니다.

Recommendation

문장으로 에셋 추천받기

화면 하단의 입력창은 원하는 장면을 자연어로 설명하는 영역입니다. 예를 들어 “수원 화성처럼 만들어줘”와 같이 입력하면 프로그램이 관련 에셋을 찾고, 가능한 경우 장면에 배치하는 흐름을 제공합니다.

  1. 장면 설명 입력

    하단 입력창에 만들고 싶은 분위기나 필요한 에셋을 문장으로 입력합니다.

  2. 전송

    전송 버튼을 누르면 프로그램이 입력 문장을 기준으로 관련 에셋을 찾습니다.

  3. 추천 결과 확인

    추천된 에셋이 표시되면 이름과 썸네일을 확인하고 필요한 항목을 선택합니다.

  4. 배치 후 조정

    자동으로 배치된 항목은 오른쪽 배치 편집 패널에서 위치, 회전, 크기를 다시 맞춥니다.

Capture

현재 화면 저장하기

조작 툴바의 캡처 버튼 또는 P 키를 사용하면 현재 장면을 이미지로 저장할 수 있습니다. 저장된 이미지는 배치 결과를 공유하거나 작업 과정을 기록할 때 사용할 수 있습니다.

권장 사용 흐름

원하는 프리셋을 고르고, 에셋을 배치한 뒤, 카메라 시점을 조정하고 캡처하세요. 화면 중앙에 건축물과 배치한 소품이 함께 보이도록 맞추면 결과를 확인하기 쉽습니다.

Troubleshooting

문제 발생 시 확인할 사항

모듈 라이브러리에 썸네일이 보이지 않습니다.

화면이 처음 열리는 동안에는 일부 이미지가 늦게 표시될 수 있습니다. 잠시 기다린 뒤에도 비어 있다면 프로그램에 포함된 에셋 리소스 상태를 확인해야 합니다.

화면이 깨져 보이거나 글자가 이상하게 표시됩니다.

폰트 에셋과 TextMesh Pro 리소스가 정상 import되었는지 확인하세요. 경고가 계속되면 Console에서 누락된 문자 또는 폰트 이름을 확인합니다.

추천 배치 기능이 작동하지 않습니다.

입력 문장을 더 구체적으로 작성해보세요. 계속 결과가 나오지 않으면 추천 기능 설정이 준비되어 있는지 확인해야 합니다.

배치한 에셋을 다시 찾기 어렵습니다.

Hierarchy에서 선택하거나, 뷰포트에서 에셋을 클릭한 뒤 F 키로 해당 위치에 초점을 맞춥니다.

이동이나 회전 조작이 어렵습니다.

오른쪽 배치 편집 패널의 숫자 입력을 먼저 사용해보세요. 마우스 조작보다 정확하게 값을 맞출 수 있습니다.
